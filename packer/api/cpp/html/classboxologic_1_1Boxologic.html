auto pack ( ) -> std::pair< std::shared_ptr< hiswill::packer::Wrapper >, std::shared_ptr< hiswill::packer::InstanceArray >>

Pack instances to the wrapper.

The pack is an adaptor method between Packer and boxologic. It encodes data from Packer, deducts the best packing solution decodes the optimization result and returns it.

The optimization result is returned as a Pair like below:

  • first: The wrapper with packed instances.
  • second: Left instances failed to pack by overloading.
Returns
A pair of wrapper with packed instances and instances failed to pack by overloading.

std::map<double, double> layer_map
private

All different lengths of all box dimensions along with evaluation values.

In other word, the layer_map stores those entries; each Boxbox's length on each axis as a key (width, height or length) and evaluation value as a value. The evaluation value means sum of minimum gaps between the key and other boxes' width, height and length

FOR i to box_array.size() WHILE key IN width, length and height in box_array[i] BEGIN value := 0 FOR j to box_array.size() value += min ( abs(key - box_array[j].width), abs(key - box_array[j].height), abs(key - box_array[j].length) ) layer_map.insert({key, value}); END

  • key: A dimension value
  • value: Evaluation weight value for the corresponding key.
