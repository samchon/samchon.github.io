Interface ICommunicator

An interface taking full charge of network communication.

ICommunicator is an interface for communicator classes who take full charge of network communication with external system, without reference to whether the external system is a server or a client.

Whenever a replied message comes from the external system has arrived, the message will be converted to an Invoke class and will be shifted to the listener's {@link IProtoco.replyData replyData()} method.
