Detailed Description

Travel을 위한 K-means 알고리즘

각 지점(Branch)의 좌표들을 수 개의 k-means 구획으로 나누고, 이들, 최적화된 구획의 순서에 따라 Travel의 정렬 순서 역시 클러스터의 정렬에 따라 재배열함으로써 유전자 알고리즘에 쓰일 초기 데이터 셋을 구성합니다.

void KMeans::cluster ( )

최적의 K-means 클러스터를 구성

K-means 구획의 도출 과정

  1. 임의의 중앙점 및 Cluster N개를 생성합니다.
  2. 각 좌표들을 가장 가까운 Cluster로 할당합니다.
  3. 각 Cluster의 중앙점을 각 Cluster 내 원소들 좌표의 평균치로 이동합니다.
  4. 각 Cluster 내부의 원소(Branch) 및 중앙점에 변화가 없다면, 종료합니다. 그렇지 않다면, 2~3의 과정을 계속 반복합니다.
