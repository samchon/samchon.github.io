Detailed Description

지점의 위치 및 해당 지점에 관한 정보

Branch는 각 지점의 경위도 좌표(GeometryPoint) 및 해당 지점의 정보를 보유하고 있습니다.

더불어 Branch의 배열 순서는 최적화의 대상으로써, 유전자 알고리즘적 관점에서는 염기서열(Scheduler) 내의 하나의 염기정보(Branch)에 해당합니다.

Note

Branch에 관한 상세사항은 아직 결정되지 않았습니다. 즉, 추후에 변경의 여지가 있습니다.
